Profil

Mahasiswi Aktif Jurusan Teknik Informatika Universitas Palangka Raya, memiliki minat kuat dalam pengembangan perangkat lunak dan teknologi informasi. Saya mempelajari pemrograman dengan bahasa PHP, Java, dan C++, serta pengalaman dalam mengembangkan aplikasi web menggunakan HTML, CSS. Selain itu, saya juga memiliki pemahaman dasar tentang basis data SQL. Saya aktif dalam organisasi kampus serta aktif dalam berbagai kegiatan akademik kampus, seperti kerja kelompok dan proyek tugas mata kuliah. Dapat dengan mudah beradaptasi di lingkungan baru, berkomunikasi dengan baik, bekerja sama dengan tim dan mampu menyelesaikan masalah dengan pengalaman di organisasi dan berbagai program kegiatan kampus.
